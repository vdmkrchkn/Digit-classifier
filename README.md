Digit classifier
====
Реализованы алгоритмы определения линейной разделимости классов (обобщенный градиентный 
спуск), Козинца - поиск оптимальной разделяющей прямой и порога (Image -> Learn), 
распознавание множества классов изображений - треугольники, цифры - (Image -> Recognize),
зашумление сетчатки (Image -> Noise). Сдвиг фигуры по сетчатке осуществляется стрелками 
на клавиатуре. Поворот фигуры - клавишами NUM4/NUM6.
