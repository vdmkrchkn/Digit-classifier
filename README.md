Image generator
====
Создание, открытие и сохранение изображения. При создании (File -> New) необходимо выбрать
размер изображения, при правильной настройке отображается треугольник, направление которого
задается радиокнопками. Сдвиг фигуры осуществляется стрелками на клавиатуре. Поворот фигуры
клавишами NUM4/NUM6. Зашумление изображения (Image -> Noise).
Реализованы алгоритмы определения линейной разделимости классов (обобщенный градиентный 
спуск), Козинца - поиск оптимальной разделяющей прямой и порога (Image -> Learn), 
классификации изображений (Image -> Recognize).
